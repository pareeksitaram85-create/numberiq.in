TDS return correction online (TRACES)

SRWritten by a Chartered Accountant NumberIQ Editorial Last reviewed June 2026 5 min read

A filed TDS return is corrected by filing a correction statement on the TRACES portal using the latest consolidated (conso) file. Corrections can add or modify challans, fix deductee PAN and amounts, and resolve short-deduction or short-payment defaults flagged in the justification report.

On this pageWhen a correction is needed · The process · Correction categories · Practical points

When a correction is needed

  • Wrong or invalid PAN of a deductee (causes the payee to lose credit).
  • Wrong challan details or unconsumed challan.
  • Short deduction / short payment / late-payment defaults.
  • Adding deductees or challans missed in the original return.

The process

  1. Download the conso file and the justification report from TRACES.
  2. Open it in the RPU/utility, make the corrections (PAN, challan, amount).
  3. Validate with the FVU and generate the correction file.
  4. Upload the correction statement (online on TRACES or at a TIN-FC).

Correction categories

TypeUsed for
Add challan / book entryAdd a missed challan
Update challanCorrect challan details
Update / add deducteeFix PAN, amount, section
PAN correctionCorrect an invalid deductee PAN

Practical points

  • Resolve defaults promptly to stop the payee's credit being blocked in 26AS.
  • Pay any short deduction/interest by challan before filing the correction.
  • There is no time limit bar for most online corrections, but act early.
  • See also fixing a 26AS mismatch.

Frequently asked questions

How do I correct a TDS return?

Download the conso file from TRACES, make the corrections in the utility, validate with the FVU, and upload a correction statement on TRACES.

Can I correct a wrong PAN in a filed TDS return?

Yes, through a PAN correction in the correction statement; this restores the deductee's TDS credit in Form 26AS.

Is there a deadline to file a TDS correction?

Most online corrections can be filed without a strict time bar, but it is best to correct promptly so the payee's credit is not delayed.

Key takeaways

  • Correct TDS returns via a correction statement on TRACES.
  • Use the conso file + justification report to identify defaults.
  • Fix PAN, challan and amount errors; pay shortfalls first.
  • Prompt correction restores the payee's 26AS credit.
